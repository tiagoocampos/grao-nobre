# Projeto Grão Nobre

Uma mini loja virtual de cafés desenvolvida para praticar **backend com
Node.js, Express e MySQL**.

O projeto simula funcionalidades básicas de um e-commerce, como listagem
de produtos e carrinho de compras.

------------------------------------------------------------------------

## Tecnologias utilizadas

-   Node.js
-   MySQL



-----------------------------------------------------------------------

## Funcionalidades atuais

-   Listar produtos
-   Buscar produto por ID
-   Adicionar produto ao carrinho
-   Visualizar itens do carrinho

------------------------------------------------------------------------

## Banco de dados

O projeto utiliza MySQL com as seguintes tabelas:

-   produtos
-   carrinho
-   carrinho_itens

------------------------------------------------------------------------

## Como executar o projeto

### 1 - Clonar o repositório

    git clone https://github.com/tiagoocampos/grao-nobre.git

------------------------------------------------------------------------

### 2 - Instalar as dependências

    npm install

------------------------------------------------------------------------

### 3 - Configurar o banco de dados

Crie um banco chamado:

    grao_nobre

Depois crie as tabelas necessárias.

------------------------------------------------------------------------

### 4 - Iniciar o servidor

    npm run dev

O servidor iniciará em:

    http://localhost:3000

------------------------------------------------------------------------

## Objetivo do projeto

Este projeto foi criado com objetivo de **aprender conceitos de
backend**, como:

-   criação de APIs REST
-   conexão com banco de dados
-   organização de código em controllers e routes
-   manipulação de dados com SQL
-   lógica de carrinho de compras
